SQS Source

See SQS Queue sample for an example of how to use the SQS Queue source.

Syntax

Sqs: String|Expression
BatchSize: Int|Expression

Properties

BatchSize

The BatchSize attribute specifies the maximum number of messages to fetch from the SQS queue. The value must be in the range from 1 to 10.

Required: No

Type: Int or Expression

Sqs

The Sqs attribute specifies the name of a resource parameter of type AWS::SQS::Queue that the Lambda function fetches messages from.

Required: Yes

Type: String or Expression
