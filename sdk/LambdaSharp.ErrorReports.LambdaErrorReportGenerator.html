Class LambdaErrorReportGenerator

The LambdaErrorReportGenerator class is used to create LambdaErrorReport instances. Each Lambda function is initialized with an instance of LambdaErrorReportGenerator class using its configuration and runtime settings. Use the ErrorReportGenerator property to access the initialized LambdaErrorReportGenerator instance in the Lambda function.
