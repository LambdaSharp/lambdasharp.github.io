Class ALambdaFunction

ALambdaFunction is the abstract base class for all AWS Lambda functions. This class takes care of initializing the function from environment variables, before invoking ProcessMessageStreamAsync(Stream).

ILambdaLogLevelLogger.Log(LambdaLogLevel, Exception, String, Object[])

Log a message wit the given severity level. The format string is used to create a unique signature for errors. Therefore, any error information that varies between occurrences should be provided in the arguments parameter.

NOTE: Nothing is logged if both format and exception are null.
