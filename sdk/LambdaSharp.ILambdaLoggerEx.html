LogErrorAsInfo(ILambdaLogger, Exception)

Log an exception as an information message. This message will only appear in the log and not be forwarded to an error aggregator.

NOTE: Only use this method when the exception has no operational impact. Otherwise, either use LogError(ILambdaLogger, Exception) or LogErrorAsWarning(ILambdaLogger, Exception).
